# 研究与发现

## 背景与目标

开发一个独立的视频下载工具软件，支持 YouTube、Bilibili 和 m3u8 直链下载视频，需要图形界面可独立运行。

## 技术实现要点

### 打包时内置工具

```
VideoDownloader/
├── bundled/                 # 内置工具（打包时包含）
│   ├── ffmpeg/             # ffmpeg 可执行文件
│   │   ├── ffmpeg.exe
│   │   └── ffprobe.exe
│   └── deno/                # Deno 运行时
│       └── deno.exe
```

- ffmpeg: 从 `D:/Software/ffmpeg-master-latest-win64-gpl/bin/` 获取
- Deno: 从 `https://github.com/denoland/deno/releases` 下载 Windows 版本

### 路径优先级
1. 用户自定义路径（如果设置中选择了"自定义"）
2. 内置路径（默认）
3. 系统 PATH（备选）

### Cookie 管理

| 平台 | Cookie 必要性 | 过期处理 |
|------|--------------|----------|
| YouTube | 必须（会员/地区限制） | 界面提示重新导入 |
| Bilibili | 必须（大会员） | 界面提示重新导入 |
| m3u8 | 不需要 | 无 |

### 项目结构

```
VideoDownloader/
├── core/                    # 核心逻辑
│   ├── downloader.py       # yt-dlp 包装器
│   ├── ffmpeg_handler.py   # ffmpeg 包装器（合并+转码）
│   ├── link_parser.py       # 链接解析与平台识别
│   ├── cookie_manager.py    # Cookie 管理
│   └── task_manager.py      # 下载任务管理
├── ui/                      # PyQt6 界面
│   ├── main_window.py       # 主窗口
│   ├── transcode_dialog.py  # 转码对话框
│   ├── settings_dialog.py  # 设置对话框
│   └── widgets/             # 自定义组件
├── utils/                   # 工具函数
├── resources/              # 资源文件
├── cookies/                 # Cookie 存储（用户数据）
├── config/                 # 配置文件
├── bundled/                 # 内置工具（ffmpeg、deno）
├── main.py                 # 入口
└── requirements.txt        # 依赖
```

## 已验证的技术方案

1. **yt-dlp**：支持 YouTube、Bilibili，带 Cookie 认证
2. **ffmpeg**：支持 m3u8 HLS 下载，支持格式转换
3. **Deno**：YouTube 签名验证需要
4. **Cookie**：Netscape 格式，EditThisCookie 导出
